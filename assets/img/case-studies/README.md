# Case Study Images Setup Guide

This directory contains images for the portfolio case studies section.

## Required Images

You need to add the following three images to this directory:

### 1. zone7.jpg
- **Website**: https://zone7.co
- **Technology**: Divi 5
- **Recommended Size**: 800x600px or similar aspect ratio
- **How to get**: Take a screenshot of the Zone7 homepage and save it as `zone7.jpg`

### 2. wpgigs.jpg
- **Website**: https://wpgigs.com/
- **Technology**: Full Site Editing with Gutenberg blocks
- **Recommended Size**: 800x600px or similar aspect ratio
- **How to get**: Take a screenshot of the WPGigs homepage and save it as `wpgigs.jpg`

### 3. cavenaughmedia.jpg
- **Website**: https://cavenaughmedia.com/
- **Technology**: Elementor
- **Recommended Size**: 800x600px or similar aspect ratio
- **How to get**: Take a screenshot of the Cavenaugh Media homepage and save it as `cavenaughmedia.jpg`

## How to Add Images

1. Visit each website in your browser
2. Take a full-page screenshot or screenshot of the hero section
3. Crop/resize the image to approximately 800x600px (or maintain 4:3 aspect ratio)
4. Save the images with the exact filenames listed above
5. Place them in this directory: `/wp-content/themes/anisur-portfolio/assets/img/case-studies/`

## Image Optimization Tips

- Use JPG format for better compression
- Optimize images using tools like TinyPNG or ImageOptim
- Keep file size under 200KB for better performance
- Ensure images are sharp and represent the website well

## Fallback

If images are not added, the case study cards will display with a dark background color as a fallback.
